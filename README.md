## Commands

### Installation

```
git clone https://github.com/Block-Chain-Valley/bv-contract-boilerplate.git
```

### Build Contracts

```
yarn build
```

### Test Contracts

```
# 테스트 결과 출력
yarn test

# 테스트 결과 및 커버리지 결과 출력
yarn coverage
```
